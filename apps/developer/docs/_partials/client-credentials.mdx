## Authentication

1. **Create a Whitelabel application**. See [Getting Started](./getting-started#create-an-application) for instructions.

2. **Get your credentials** — after registration you receive a `client_id` and `client_secret` from the application settings.

3. **Request tokens** by posting your credentials to:

<Endpoint method="POST" path="/auth/token" href="https://docs.monerium.com/api/#tag/auth/operation/auth-token" />

<Tabs>
  <TabItem value="node" label="Node.js">

```js
const response = await fetch('https://api.monerium.dev/auth/token', {
  method: 'POST',
  headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
  body: new URLSearchParams({
    grant_type: 'client_credentials',
    client_id: 'YOUR_CLIENT_ID',
    client_secret: 'YOUR_CLIENT_SECRET',
  }),
});

const { access_token, expires_in } = await response.json();
// {
//   access_token: "EoWmpc2uSZar6h2bKgh",
//   expires_in: 3600,
//   token_type: "Bearer"
// }
```

  </TabItem>
  <TabItem value="go" label="Go">

```go
import (
  "encoding/json"
  "net/http"
  "net/url"
)

type TokenResponse struct {
  AccessToken string `json:"access_token"`
  ExpiresIn   int    `json:"expires_in"`
  TokenType   string `json:"token_type"`
}

data := url.Values{
  "grant_type":    {"client_credentials"},
  "client_id":     {"YOUR_CLIENT_ID"},
  "client_secret": {"YOUR_CLIENT_SECRET"},
}

resp, err := http.PostForm("https://api.monerium.dev/auth/token", data)

var tokens TokenResponse
json.NewDecoder(resp.Body).Decode(&tokens)
// {
//   AccessToken: "EoWmpc2uSZar6h2bKgh",
//   ExpiresIn:   3600,
//   TokenType:   "Bearer"
// }
```

  </TabItem>
</Tabs>

4. **Store the access token** and include it as `Authorization: Bearer <token>` on every API request. It expires after 1 hour — when it does, repeat this request to get a new one.

:::warning Keep credentials and tokens server-side
Your backend should handle the token exchange and make all Monerium API calls. Never expose your `client_secret` or access token to frontend code.
:::
