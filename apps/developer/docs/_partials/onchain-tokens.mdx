## On-chain transfers and Monerium tokens

Monerium issues regulated e-money as programmable blockchain tokens (EURe, GBPe), compatible with standard DeFi protocols.

- **Contracts:** Open source and audited — [monerium/smart-contracts](https://github.com/monerium/smart-contracts)
- **Token addresses:** Listed on the [Tokens](./tokens) page.
- **ERC-20:** Standard token interface — compatible with wallets, exchanges, and DeFi protocols out of the box.
- **ERC-2612 (Permit):** Enables gasless approvals. Users sign off-chain, eliminating a separate approval transaction and reducing friction. [Tutorial →](https://github.com/monerium/smart-contracts/blob/main/docs/permit.md)
