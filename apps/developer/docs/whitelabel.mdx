# Whitelabel

Build under your own brand using the Monerium API. Your users never interact with Monerium interfaces, and each customer gets a dedicated IBAN and profile exclusive to your application.

## Integration flow

1. **[Authenticate](#authentication):** Obtain an access token using Client Credentials.
2. **[Create a profile](#create-a-profile):** Register a new customer with `POST /profiles`.
3. **[Onboard the customer](#onboarding):** Submit KYC/KYB data via KYC Sharing or KYC Reliance. Monitor approval via webhooks.
4. **[Link a wallet](#link-wallet):** Associate a blockchain address with the profile.
5. **[Issue an IBAN](#iban):** Provision a dedicated IBAN once the profile is approved and a wallet is linked.
6. **[Send and receive payments](#orders):** Place redeem orders for outgoing SEPA payments and bridging. Incoming payments are handled automatically.
